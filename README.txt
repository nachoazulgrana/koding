Este repositorio, es una prueba en donde desde la web de koding.com, creamos todos los archivos y utilizamos su propia consola para manejar git y compilar Stylus, sin la necesidad de la consola del sistema operativo, ni tampoco ningún editor de código. Publicando directamente desde la web al repositorio.
En el caso de estar usando Windows por algún motivo, no necesitamos utilizar su consola, ni instalar softwares como GitBush
